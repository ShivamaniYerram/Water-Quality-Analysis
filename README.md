# Water-Quality-Analysis
End-to-end Data Analytics project on Water Quality using Python (EDA) and Power BI Dashboard.
# Water Quality Analysis Dashboard
## Project Overview

This project focuses on analyzing water quality data collected from CPCB (Central Pollution Control Board). The data was extracted from PDF format, converted into Excel, cleaned using Python, and visualized using Power BI.

### Objective

Analyze water quality parameters across different states

Identify polluted regions

Build an interactive dashboard for insights

### Tools & Technologies

Python (Pandas, NumPy, Matplotlib)

Google Colab

Power BI

Excel

## Project Workflow
### 1.Data Collection

Source: CPCB Reports (PDF)

Converted PDF → Excel

### 2️.Data Cleaning (Python - Colab)

Removed missing values

Handled inconsistent state names

Cleaned special characters (?, missing values)

Converted data into structured format

### 3️.Exploratory Data Analysis (EDA)

Checked distribution of:

pH levels

BOD (Biochemical Oxygen Demand)

DO (Dissolved Oxygen)

Identified trends and anomalies

### 4️.Dashboard Creation (Power BI)

Built interactive dashboard with:

KPI Cards (Monitoring Locations, Avg pH, DO, BOD)

Pollution Category Distribution

Year-wise comparison (2020 vs 2021)

State-wise analysis

pH Min vs Max comparison
## Key Insights

Majority of water bodies fall under Clean category (~38%)

Some states show high BOD levels indicating pollution

DO levels vary significantly across regions

Slight variation observed between 2020 and 2021
